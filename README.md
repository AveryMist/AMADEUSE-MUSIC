# AMADEUSE MUSIC
Une application multiplateforme de streaming musical développée avec Flutter (Android, Windows).

*A cross-platform music streaming app developed with Flutter (Android, Windows).*

## Fonctionnalités / Features
* Lecture de musique depuis YouTube/YouTube Music / *Play songs from YouTube/YouTube Music*
* Cache des chansons pendant la lecture / *Song caching while playing*
* Support radio / *Radio feature support*
* Musique en arrière-plan / *Background music*
* Création de playlists et favoris / *Playlist creation & bookmarks*
* Favoris artistes et albums / *Artist & Album bookmarks*
* Import via partage depuis YouTube/YouTube Music / *Import via sharing from YouTube/YouTube Music*
* Contrôle de la qualité de streaming / *Streaming quality control*
* Téléchargement de chansons / *Song downloading*
* Support multilingue / *Multi-language support*
* Ignorer les silences / *Skip silence*
* Thème dynamique / *Dynamic theme*
* Interface utilisateur flexible / *Flexible UI (Bottom/Side navigation)*
* Support égaliseur / *Equalizer support*
* Support Android Auto / *Android Auto support*
* Paroles synchronisées et texte simple / *Synced & Plain lyrics*
* Minuteur de sommeil / *Sleep timer*
* Sans publicité / *No advertisements*
* Aucune connexion requise / *No login required*

## Dépannage / Troubleshooting
* **Problèmes de notification ou arrêt de la lecture** : Activez l'option "Ignorer l'optimisation de la batterie" dans les paramètres
* *Notification issues or playback stopped by system optimization*: Enable "Ignore battery optimization" in settings
